Pricing architecture

Free

Open protocol + SDK adoption

Access protocol repositories, SDK source, and self-managed integration workflows at no software-license cost.

  • SDK download and local use
  • Public docs and examples
  • Community support channels
Usage-based

Hosted services

Metered services for teams using managed infrastructure instead of running every component in-house.

  • Hosted relay/API throughput
  • Managed observability surfaces
  • Scalable runtime capacity
Contracted

Enterprise and sovereign programs

Commercial agreements for long-horizon deployments requiring SLAs, controls, and dedicated support.

  • Compliance and architecture workshops
  • Production onboarding and governance
  • Named support and escalation paths
No charge is attached to simply obtaining the SDK. Commercial billing starts with hosted usage or contracted scope.
