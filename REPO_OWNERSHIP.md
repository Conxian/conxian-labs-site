# Repo ownership

## Purpose

`conxian-labs-site` is the public narrative and entry surface for the Conxian portfolio.

## This repo owns

- public-facing site content
- top-level ecosystem narrative
- entrypoint communication for builders, partners, and ecosystem visitors

## This repo does not own

- canonical integration logic
- shared-core ownership
- protocol identity
- reference-client behavior

## Boundary rule

This repo should present the portfolio clearly and consistently, but not become a hidden home of platform logic.

## Strategic role

Supporting repo.
