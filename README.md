Official landing page for Conxian Labs (Pty) Ltd - Sovereign Infrastructure for the Bitcoin Renaissance.

## Purpose

Provide a public-facing overview of Conxian Labs, its flagship products, and the Conxian ecosystem narrative.

## Status

Active. Content and visuals evolve as the Conxian stack and positioning mature.

## Audience

- The general public learning what Conxian is.
- Partners evaluating integrations, services, and product direction.
- Contributors looking for an ecosystem-level entry point.

## Relationship to the Conxian stack

- Public entry point that points users toward Conxius Wallet, Conxian Finance, and Conxian Gateway.
- Complements the BOS OpenSpec documentation and technical repositories.

## Features
- **Sovereign Aesthetic:** Implements the "Earthy Corporate Finance" theme (Deep Green & Gold).
- **Interactive Protocol Visualization:** A "Root to Leaf | Leaf to Root" diagram representing the flow of Vision and Settlement across the Conxian ecosystem.
- **Strategic Triad:** Showcases core products:
  - **Conxius Wallet:** Sovereign Access.
  - **Conxian Finance:** Automated Settlement.
  - **Conxian Gateway:** Institutional Bridge.

## Technical Details
- **Architecture:** Pure static HTML/CSS/JS for maximum sovereign delivery.
- **Visualization:** Interactive SVG with dynamic particle flow systems.
- **Mobile Responsive:** Optimized for both desktop and mobile viewing.
