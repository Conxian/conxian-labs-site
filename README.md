## Ecosystem Vision
Conxian-Labs builds sovereign infrastructure that bridges Bitcoin (L1) and Stacks (L2) to enable high-performance, hardware-secured, and automated business settlement (SAB). This repository contains the static landing page representing the unified Conxian Protocol.

## Portfolio Taxonomy
The Conxian ecosystem is categorized by its functional role:
- **Product UI:** Direct user interaction layers ([Conxian_UI](https://github.com/Conxian/Conxian_UI)).
- **Platform/Core:** Master orchestration ([conxius-platform](https://github.com/Conxian/conxius-platform)).
- **Wallet:** Sovereign access and TEE-secured vaults ([conxius-wallet](https://github.com/Conxian/conxius-wallet)).
- **Gateway/Rails:** Institutional bridges and unified APIs ([conxian-gateway](https://github.com/Conxian/conxian-gateway)).
- **SDK:** Cross-platform cryptographic primitives ([lib-conclave-sdk](https://github.com/Conxian/lib-conclave-sdk)).
- **Deployment:** GUI/TUI management for Stacks L2 ([stacksorbit](https://github.com/Conxian/stacksorbit)).

## Technical Architecture
- **Tech Stack:** Static HTML5 / CSS3 / Vanilla JavaScript.
- **Design System:** "Ivory Foundation" — High-contrast, institutional, and earthy aesthetic.
- **Typography:** 'JetBrains Mono' (Engineering Precision).
- **Visualization:** SVG-based "Root to Leaf" particle flow representing Vision and Settlement.
