# Conxian-Labs Site

This repository is the public narrative and entry surface for the Conxian portfolio.

## Role

It exists to present Conxian clearly to builders, partners, and ecosystem visitors.

## Owns

- public-facing site content
- top-level portfolio and ecosystem narrative
- outward-facing messaging and entrypoint communication

## Does not own

- canonical integration logic
- shared-core ownership
- protocol identity
- reference-client behavior

## Relationship to the portfolio

This repo should describe the platform clearly and consistently while leaving implementation concerns to the strategic technical repos.
